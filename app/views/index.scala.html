Better GOVHK API

This site aims to create a better way to view the existing traffic data provided by HK Government

With some analysis on the current traffic data in GOV DataOne API, it's found that some information is missing, especially the GEO coordination data in the traffic links, I have managed to create this API for all of us to get more data.

Data from HK GOV

you will get the latest traffic xml data from hk gov at: http://resource.data.one.gov.hk/td/speedmap.xml

Data from BETTER GOVHK API

you will get from following api: http://govapi.amoebaconsulting.net/api/traffic/speedmap/latest

About this project

I have made this project open source on github, anyone is welcome to join and make this API even better https://github.com/yeukfung/dataoneapi
